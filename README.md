# 図書委員当番くん (Tosho-in Wariate-kun)

図書委員当番くんは、学校の図書委員の当番管理をサポートするウェブアプリケーションです。

## 機能

- 図書委員の当番スケジュール管理
- ユーザー認証（ログイン・ログアウト）
- 管理者機能

## 技術スタック

- フロントエンド: Next.js 15, React 19, TailwindCSS
- 言語: TypeScript

## 開発環境のセットアップ

```bash
# リポジトリをクローン
git clone https://github.com/ysogabe/tosho-in-wariate-kun.git
cd tosho-in-wariate-kun

# フロントエンドの依存関係をインストール
cd frontend
npm install

# 開発サーバーを起動
npm run dev
```
